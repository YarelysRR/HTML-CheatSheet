HTML ELEMENTS

Tags Name Description Example
<html> HTML The <html> element is the root element and it defines the whole HTML document. It has a start tag < html> and an end tag <html>. Then, inside the <html> element there is a <body> element: <!DOCTYPE html> <html> <body> <h1> My First Heading </h1> <p> My first paragraph. </p> </body> </html>.
<body> Body The <body> element defines the document's body. It has a start tag <body> and an end tag <body>. Then, inside the <body> element there are two other elements: <h1> and <p>.
<head> Head The < head > element contains meta information about the HTML page < head > <title> Page Title </title> </ head >
<h1> - <h6> Headings HTML headings are titles or subtitles that you want to display on a webpage.HTML headings are defined with the <h1 gt to <h6> tags. <h1> defines the most important heading. <h6> defines the least important heading.

<p> Paragraph A paragraph always starts on a new line, and is usually a block of text. <p> This is a paragraph. <p>
<a> Links HTML links are defined with the < a > tag <a href="https://www.w3schools.com"> This is a link </a>
<img> Image HTML images are defined with the <img> tag. The source file (src), alternative text (alt), width, and height are provided as attributes: <img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">
